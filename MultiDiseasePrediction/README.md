# 🏥 MedPredict AI

An Explainable AI based Multi-Disease Prediction System developed using Machine Learning and SHAP Explainability.

### 👨‍💻 Developed by Sumit Choudhary
Final Year B.Tech CSE Major Project

---

## 📌 About Project

This project predicts the risk of:
- Diabetes
- Heart Disease
- Kidney Disease

using trained Machine Learning models like Random Forest and XGBoost.

The system also provides SHAP based explainability to show which health features increase or decrease disease risk.

---

## 🚀 Features

- User Login & Registration
- Disease Risk Prediction
- Combined Multi-Disease Analysis
- SHAP Explainability
- Prediction History
- Model Performance Comparison
- PDF Report Download
- Responsive Healthcare Dashboard
